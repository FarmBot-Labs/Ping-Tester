# Ping Tester

This is an example application that helps developers ping their bots remotely.

It's also a good example of how to write non-Farmware extensions.

![](screenshot.png)

![](screenshot2.png)

# Installation

0. Clone repo
0. `npm install`

#  Run

0. `npm  start`

Open `http://localhost:1235/` and enter a  device token.
